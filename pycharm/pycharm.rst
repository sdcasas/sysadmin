pycharm
=======

shortcuts
---------

* shift + shift         --> Search
* alt + F12             --> open shell
* ctrl + shift + n      --> go to file
* ctrl + e              --> recent files
* ctrl + f4             --> close file (tab acvive)


# run and debuging
* F8                    --> run next line
* F9                    --> next breakpoint
* shift + F10           --> run
* shift + F9            --> run debug
